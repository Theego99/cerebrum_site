# Studio Cerebrum — Landing Page

A single-page, static landing page for Studio Cerebrum. Built with plain HTML, CSS, and vanilla JavaScript.

## Project Info

- **URL:** https://studiocerebrum.com
- **Tech:** Static HTML/CSS/JS, no build step
- **Hosting:** GitHub Pages
- **Reference:** [OK-RM](https://www.ok-rm.co.uk)

## Local Development

```bash
python3 -m http.server 8000
# -> open http://localhost:8000
```

## Deploy

Push to `main` branch; changes deploy automatically via GitHub Pages.
